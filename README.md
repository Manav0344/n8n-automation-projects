# Lead Capture & Email Automation

## Project Overview
This project demonstrates an automated lead capture workflow built using n8n.  
The system collects form submissions, stores user data in Google Sheets, and sends automated email notifications.

The automation helps reduce manual data handling and improves lead management efficiency.

## Tools Used
- n8n
- Google Sheets
- Gmail

## Workflow Process
1. User submits a form.
2. The form data is received by n8n.
3. The lead information is automatically stored in Google Sheets.
4. An automated email notification is sent using Gmail.

## Features
- Automated lead capture
- Data storage in Google Sheets
- Automated email notification
- Workflow automation using n8n

## Use Case
This workflow can be used for:
- Website contact forms
- Lead generation systems
- Customer inquiry management

## Example Workflow
Form Submission → n8n Workflow → Store Data in Google Sheets → Send Email Notification
